Variable:To store a value we need a variable.

Declaring A Variable

While declaraing a variable there are some naming conventions. That you can use only $ , _ , letters and numbers. But numbers can be use after the letters. For Example: pratham007.

Note: We also cannot use reserved keywords of JavaScript. But only one keyword can be used i.e name.But it's not a good practice to use name.

Datatypes

In JavaScript a value is always either a object or a primitive.

Objects

Syntax:
let me = {
name: '2Pac';
song:'Dreams' ;
}

Primitive

Syntax:
let firstName = 'Eminem'

There are 7 Primitive Datatypes:

  • Number: Float point numbers (which means they are always in decimal even if we write 23 i.e 23.00) used to store decimals and integers.
    Sytnax: let MobileNo = "123465790";
  • String: Sequesnce of characters, used for text.
    Syntax: let name = "Drake";
    Note: Always use single or double quotes to declare a string.
  • Boolean: Logical type that can be either true or false.
    Syntax: let bike = true;
  • Undefined: Value taken by variable that is not yet defined
    Syntax: let bike;
  • Symbol (ES2015) : Value i.e unique and cannot be changed.
  • BigInt: Larger integers that the numbers type can't hold.

JavaScript Dynamic Typing

We do not have to manually define the data type of the value ( Value has type ) stored in a variable. Instead data types are determined automatically.

typeof() operator

It is used to see the type of value.
Syntax: console.log(typeof(69));
Result: It will print number.

let, const & var

let: It is used when we want to change the value of variable later in our program.
It is also used to declare a Undefined (empty variable).

const: To declare a variable whose value cannot be changed.
Reassigning such a variable leads to Type Error.
We cannot use const to declare a empty variable.

Note: Always use const by default, only use let when the value is going to changed in future.

Var: It is the old way of declaring the variable. Works the same way as let.
